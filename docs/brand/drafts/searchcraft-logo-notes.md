# Searchcraft Logo Draft Notes

The production mark is based on the third draft direction: a compact
search/compass symbol with a small blue accent. It follows `DESIGN.md` by
using neutral geometry, a single interaction-blue accent, no gradients, no
glow, and no illustrative decoration.

Runtime assets live in `public/brand/`. The app renders the same geometry
through `src/components/brand-logo.tsx` so dark-mode foreground colors can use
the app tokens instead of a fixed black image.
